## Proposal

This project aims to be an extension (or fork) of ReactStrap to build some new components (e.g. Timeline, Card flavors and among others).

## Installation

```
npm install
docker build -t katielly .
docker run -it --rm -v $(pwd):/code katielly bash
> cd bootstrap-4.3.1/
> npm install
> npm run dist
```
