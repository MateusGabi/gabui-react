## Proposal

This project aims to be an extension (or fork) of ReactStrap to build some new components (e.g. Timeline, Card flavors and among others).

## Installation

```
npm install
cd bootstrap-4.3.1/
npm install
```

### Generate Bootstrap build

```
cd bootstrap-4.3.1/
npm run dist
```

### Run Docz

```
npm run docz:dev
```
